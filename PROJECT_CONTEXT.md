# TechBusiness Website

## Project Overview
A modern, professional website for a technology business targeting general consumers. The website showcases professional services and features client testimonials to build trust and credibility.

## Brand Identity
- **Industry**: Technology
- **Target Audience**: General consumers
- **Style**: Modern and professional
- **Color Palette**: Blues (trust, professionalism), whites and grays (clean, modern), accent colors for CTAs

## Key Features

### 1. Professional Services
- Cloud Solutions
- Cybersecurity
- AI & Machine Learning
- Data Analytics
- Mobile Development
- Business Intelligence

### 2. Client Testimonials
- Carousel-based layout (different from services grid)
- 6 authentic testimonials
- Professional photos
- 5-star ratings
- Company affiliations

### 3. Pricing
- Single package at $29
- Stripe integration for payments
- Trust indicators
- Money-back guarantee

### 4. Contact Form
- Integrated with API endpoint
- Hidden fields for founder_id and project_id
- Success/error handling
- Professional design

## Design Principles

### Typography
- Large, impactful headings (text-6xl, text-7xl)
- Clear hierarchy
- Generous line spacing
- Inter font family

### Layout
- Full-screen hero with overlay
- 3-column grid for services
- Carousel for testimonials (different from services)
- Generous white space throughout
- Responsive design

### Colors
- Primary: Blue (#2563eb)
- Secondary: Gray tones
- Accent: Cyan, purple for variety
- Background: White to gray gradient

### Components
1. Sticky navigation with logo and CTA
2. Full-screen hero with gradient overlay
3. Services grid with icons
4. Testimonials carousel
5. Pricing section with Stripe integration
6. Contact form with validation
7. Comprehensive footer
